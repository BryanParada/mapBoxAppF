# mapbox_maps

A Flutter project with mapbox library.

## Features  
- Zoom in - zoom out
- Change style

## What did i use in this app? 

- MapBox Library 
- Custom Themes in Mapbox
- symbols
- Custom Symbols
- Change theme dynamically
